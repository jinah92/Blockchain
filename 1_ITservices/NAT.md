# NAT

## NAT(Network Address Translation)과 NAT 네트워크

---

### NAT

- 가상머신 내부 네트워크 ⇒ Host PC 외부 네트워크 단방향 통신 가능
- **Host PC 내의 가상 머신 간의 통신 불가능**
- 활용 : 가상머신 내 단순한 인터넷 사용, 파일 다운로드 등

### NAT Network

- 가상머신 내부 네트워크 ⇒ Host PC 외부 네트워크 단방향 통신 가능
- **Host PC 내의 가상 머신 간의 통신 가능**
- 활용 : 가상머신 간 상호 통신이 필요한 작업
